Controls

デフォルト: キーボード ← → で移動, SPACEで射撃, Escで戻る

HOW TO PLAY

操作

  • ← → : 機体移動
  • SPACE : ショット
  • A : 実績ウィンドウ
  • R : リスタート
  • Esc : メニューに戻る

目的

Waveを進め、敵を殲滅し、スコアを稼いで実績と報酬を獲得しましょう。

ボスWaveでは特別なパターンと攻撃が待ち構えています。報酬を活用し、最終Waveを目指してください。

ヒント

  • コンボを維持するとスコア倍率が上昇。
  • パワーアップを活用して状況を有利に。
  • 難易度によって敵挙動と報酬が大きく変化。
